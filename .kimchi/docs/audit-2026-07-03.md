### Risque critique identifié
**`.env.local` contient des credentials de production** et apparaît dans `git status` modifié. Même s’il est gitigné aujourd’hui, il a pu être commité auparavant. Il faut :
1. Supprimer `reqy-web/.env.local` de l’historique git (`git filter-repo` ou BFG).
2. Rotate tous les secrets concernés (GitHub OAuth, Google OAuth, Jina AI, `AUTH_SIGNING_SECRET`).
3. Ne jamais versionner `.env.local`.
